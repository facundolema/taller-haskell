# Taller de Algebra I #

## Test de funciones ###

**Tester** es un script (no oficial) para verificar la validez de las funciones dadas como tarea en cada clase del taller de Algebra I. Lo unico que hace **Tester** es ejecutar las funciones con diferentes par√°metros y verificar si la respuesta es la esperada. **Tester** viene en cinco formas distintas, dependiendo la plataforma en la que se quiera utilizar:
- **PowerShell** (Windows 7 / 8 / 10 / 11, PowerShell Core 6 / 7)
- **Batch** (Windows) [en proceso]
- **Bash** (Linux, macOS, *nix) [en proceso]
- **ZSH** (macOS, zsh users) [en proceso]
- **Ejecutable (C)** (ver *releases*) [en proceso]
